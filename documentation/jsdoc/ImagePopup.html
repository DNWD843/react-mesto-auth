render() → {JSX}

Публичный метод render()
Возвращает JSX-фрагмент разметки - попап с полноразмерным изображением карточки
При отрисовке компонента непосредственно в разметке производится проверка наличия данных карточки:
- если данные есть, они подставляются в соответствующие элементы, попап получает открывающий модификатор 'popup_opened'
- если данных карточки нет, вместо данных подставляются пустые строки или символ '#', а попап не получает открывающий модификатор 'popup_opened' и остается невидимым для пользователя.
