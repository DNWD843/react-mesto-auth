Module: Main

Функциональный React-компонент
Декларирует отрисовку основного контента приложения: профиль пользователя, карточки с изображениями
В компонент Main импортируются:
- api - экземпляр класса Api
- Card - компонент Card
Принимает объект параметров - props
Возвращает JSX-фрагмент разметки основного контента приложения
Parameters:
Name Type Description
props Object объект с переданными функциями-коллбэками
Properties
Name Type Description
onAddPlace function функция-коллбэк, вызывается при клике по кнопке "Добавить фото", открывает попап добавления карточки
onCardClick function функция-коллбэк, вызывается при клике по изображению карточки, открывает попап с полноразмерным изображением карточки
ПРИМЕЧАНИЕ В компоненте Main не используется. Через пропсы компонента Main пробрасывается в компонент Card.
onEditAvatar function функция-коллбэк, вызывается при клике по аватару пользователя, открывает попап редактирования аватара пользователя
onEditProfile function функция-коллбэк, вызывается при клике по кнопке "Редактировать профиль", открывает попап редактирования профиля
onCardDelete function функция-коллбэк, вызывается при клике по иконке удаления карточки, открывает попап подтверждения удаления карточки
onCardLike function функция-коллбэк, вызывается при клике по "лайку" карточки, ставит или удаляет "лайк", в зависимости от его состояния
cards Array массив объектов(карточек)
