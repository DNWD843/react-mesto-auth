Module: Card

Функциональный React-компонент
Декларирует отрисовку карточки
Принимает объект параметров - props
Возвращает JSX-фрагмент разметки - карточку, заполненный данными карточки
Parameters:
Name Type Description
props Object объект с параметрами и переданными функциями-коллбэками
Properties
Name Type Description
id String уникальный идентификатор карточки, присваивается карточке сервером.
ПРИМЕЧАНИЕ: props.id используется при отрисовке компонента Card для присвоения обязательному параметру key
link String ссылка на изображение
title String название изображения
likesQuantity Number количество лайков
owner Object объект, данные о владельце карточки
likes Array массив, содержит id всех пользователей, лайкнувших карточку
onCardClick function функция-колбэк, вызывается при клике по изображению карточки, открывает попап с полноразмерным изображением карточки
Принимает аргументом объект с названием карточки и ссылкой на изображение для дальнейшей передачи этих параметров в ImagePopup
onCardLike function функция-колбэк, вызывается при клике по "лайку" карточки, ставит/снимает "лайки", принимает аргументом объект с id карточки и массивом лайков карточки
onCardDelete function функция-колбэк, вызывается при клике по иконке удаления карточки, открывает попап подтверждения удаления, принимает аргументом объект с id карточки
