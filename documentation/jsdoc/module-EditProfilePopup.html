Module: EditProfilePopup

Функциональный React-компонент
Редактирует данные профиля пользователя
Для успешного изменения профиля пользователь должен ввести валидные данные в поля "Имя" и "О себе"
Принимает объект параметров props
Возвращает JSX-фрагмент разметки попапа редактирования профиля пользователя
Parameters:
Name Type Description
props Object объект с параметрами
Properties
Name Type Description
onClose function функция-коллбэк, закрывает попап
onOverlayClick function функция-коллбэк, вызывается при клике по оверлею, закрывает попап
onUpdateUser function функция-коллбэк, обновляет данные профиля пользователя при успешном сабмите формы редактирования профиля
isOpen Boolean индикатор состояния попапа, управляет его видимостью:
- true - попап открыт
- false - попап закрыт
isLoading Boolean индикатор состояния загрузки новых данных профиля пользователя, управляет отображением прелоадера загрузки:
- true - идет загрузка, отображется текст-прелоадер на кнопке сабмит;
- false - нет загрузки, на кнопке сабмит дефолтный текст
