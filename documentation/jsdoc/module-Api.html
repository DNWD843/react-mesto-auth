Module: Api

Класс Api
Отвечает за отправку запросов на сервер и проверку полученных ответов
Parameters:
Name Type Description
Object Object принимает в конструктор объект { URLs, headers, token }
Properties
Name Type Description
URLs Object объект, содержащий адреса для отправки запросов
Properties
Name Type Description
baseURL String базовый URL сервера
cardsURL String URL для получения/добавления карточек
userURL String URLдля получения/добавления данных пользователя
likesURL String URL для получения/добавления информации о "лайках"
avatarURL String URL для получения/добавления ссылки на аватар
headers Object объект, содержащий заголовки запросов
Properties
Name Type Description
authorization String код авторизации (токен)
token String токен для доступа на сервер
Since:
  • v.1.0.0
Source:
  • utils/Api.js, line 1
Example
const api = new Api({
URLs: {
  baseURL: 'https://mesto...',
  cardsURL: 'https://mesto.../cards/',
  userURL: 'https://mesto.../users/me',
  likesURL: 'https://mesto.../cards/likes/',
  avatarURL: 'https://mesto.../users/me/avatar'
},
headers: {
  "authorization": '<Ваш код авторизации>'
},
token: '<Ваш токен>'
});
