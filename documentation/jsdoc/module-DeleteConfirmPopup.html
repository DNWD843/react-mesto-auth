Module: DeleteConfirmPopup

Функциональный React-компонент
Запрашивает подтверждение удаления карточки
Принимает объект параметров props
Возвращает JSX-фрагмент разметки попапа подтверждения удаления карточки
Parameters:
Name Type Description
props Object объект параметров
Properties
Name Type Description
onClose function функция-коллбэк, закрывает попап
onOverlayClick function функция-коллбэк, вызывается при клике по оверлею, закрывает попап
onSubmit function функция-коллбэк сабмита попапа, удаляет карточку
isOpen Boolean индикатор состояния попапа, управляет его видимостью:
- true - попап открыт
- false - попап закрыт
isProcessing Boolean индикатор статуса процесса удаления карточки: true - карточка удаляется, false - ожидается подтверждение удаления карточки или отмена
