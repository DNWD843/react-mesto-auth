Module: InfoToolTip

Функциональный React-компонент
Попап с информационной подсказкой об успехе или ошибке регистрации в приложении
Открывается при попытках пользователя зарегистрироваться или войти в приложение.
Parameters:
Name Type Description
props Object пропсы, принимаемые компонентом
Properties
Name Type Description
name String название попапа. Не обязательный пропс. Значение по умолчанию: "info-tool-tip"
isOpen Boolean индикатор состояния попапа, управляет его видимостью:
- true - попап открыт
- false - попап закрыт
onClose function функция-коллбэк, вызывается при клике по иконке закрытия попапа, закрывает попап
onOverlayClick function функция-коллбэк, вызывается при клике по оверлею, закрывает попап
titleTextSuccess String текст подсказки об успешной регистрации. Не обязательный пропс. Значение по умолчанию: "Вы успешно зарегистрировались!"
titleTextFail String текст подсказки об ошибке регистрации или авторизации. Не обязательный пропс. Значение по умолчанию: "Что-то пошло не так! Попробуйте ещё раз."
loggedIn Boolean индикатор состояния пользователя, управляет его допуском в приложение:
- true - пользователь зарегистрирован и залогинился
- false - не зарегистрирован, не залогинился или вышел из приложения
